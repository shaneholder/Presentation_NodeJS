NodeJS is...

  • Not no Javascript.
  • JavaScript on the server
  • Asynchronous by default.
  • Not threaded
  • Node.js is an evented I/O framework for the V8 JavaScript engine. It is intended for writing scalable network programs such as web servers.. http://en.wikipedia.org/wiki/Nodejs
  • Fun!! well at least I think so.
